# A Federated Knowledge Graph Architecture for Dutch Cadastral Data

Conference Paper

[← Back to Works](../works.llms.md)

Kadaster · University of Twente

------------------------------------------------------------------------

Proposes a federated virtual knowledge graph architecture that enables interoperable querying over Dutch cadastral data sources without centralizing the underlying datasets. The architecture leverages ontology-based data access (OBDA) principles and SPARQL federation to connect heterogeneous land registration systems in a privacy-respecting manner.
